APIキーを設定

このアプリをブラウザから直接動かすには、あなた自身のAnthropic APIキーが必要です。キーはこの端末のブラウザ内にのみ保存され、Anthropic以外には送信されません。

開発者ツールを開けば読み取れる状態で保存されます。共有PCや配布用のファイルには入力しないでください。キーは console.anthropic.com の Settings → API Keys から発行できます。

開き方を変える必要があります

このファイルをfile://として直接開くと、ブラウザの制約でAnthropic APIへの通信がブロックされます(APIキーが正しくても失敗します)。ローカルの簡易サーバー経由で開けば解決します。

ターミナル(コマンドプロンプト)でこのファイルがあるフォルダに移動して、次のいずれかを実行してください:

python3 -m http.server 8000

Pythonがなければ:

npx serve .

実行したら、ブラウザで http://localhost:8000/ai_chat.html(または表示されたURL)を開いてください。ファイルをダブルクリックする方法では動きません。
